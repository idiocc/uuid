## API

The package is available by importing its default function from the `v4` file:

```js
import uuid from '@goa/uuid/v4'
```

%~%

```## v4
[
  ["options?", "UuidConfig"],
  ["buffer?", "Array|Buffer"],
  ["offset?", "number"]
]
```

Generate and return a RFC4122 v4 UUID.
