## API

The package is available by importing its default function. Only `v4` is supported.

```js
import v4 from '@goa/uuid'
```

%~%

```## v4 => string
[
  ["options?", "UuidConfig"],
  ["buffer?", "Array|Buffer"],
  ["offset?", "number"]
]
```

Generate and return a RFC4122 v4 UUID.
